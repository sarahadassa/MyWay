# MyWay — Portfólio Profissional

## Problema resolvido

Estudantes e profissionais de tecnologia não têm uma forma simples de reunir e compartilhar sua trajetória profissional. O MyWay resolve isso permitindo que qualquer pessoa crie um portfólio online em minutos, sem precisar saber programar, e compartilhe com um link único.

---

## Como executar

1. Clone o repositório
```bash
git clone https://github.com/sarahadassa/MyWay.git
```

2. Abra a pasta no VS Code

3. Clique com o botão direito no `index.html` e selecione **Open with Live Server**

> É necessário ter a extensão **Live Server** instalada no VS Code.

---

## Tecnologias usadas

- **HTML, CSS e JavaScript** — estrutura, estilo e lógica da aplicação
- **Firebase Authentication** — cadastro e login de usuários
- **Cloud Firestore** — banco de dados em nuvem
- **Cloudinary** — upload e hospedagem de imagens
- **Google Fonts** — tipografia (DM Sans e DM Serif Display)
